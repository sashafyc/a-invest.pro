## Форматы по платформам

### VC.ru (`vc.html`)
- **Объём:** 50–60% от оригинала
- **Угол:** профессиональный, аналитический. Факты, цифры, практика.
- **Как копировать:** открыть `.html` в браузере → Cmd+A → Cmd+C → вставить в редактор VC.ru
- **Форматирование:** h2 заголовки, bold, ul/li — всё сохраняется при копировании
- **Финал:** ссылка «Читать полностью» + ссылка на express-ocenka.html + подпись Александра
- **Не дублировать** первое предложение — VC.ru режет тексты с одинаковым началом

### Дзен (`dzen.html`)
- **Объём:** 60–70% от оригинала
- **Угол:** личный, storytelling. «Я видел», «У меня был клиент», «Расскажу честно».
- **Заголовок:** указан в комментарии файла — другой, не как у статьи на сайте
- **Как копировать:** так же — браузер → Cmd+A → Cmd+C → вставить в редактор Дзен
- **Финал:** ссылка на полную статью + ссылка на express-ocenka.html + подпись
- **Важно:** Дзен любит личные истории, первое лицо, конкретные случаи из практики

### TenChat (`tenchat.txt`)
- **Объём:** 400–600 символов
- **Формат:** plain text, без HTML-тегов
- **Структура:** 1 тезис → 3–5 буллетов → ссылка → хэштеги
- **Хэштеги:** #бизнес #продажабизнеса #предпринимательство #брокер #инвестиции
- **Как публиковать:** копировать текст → вставить в TenChat → проверить отображение буллетов

### Habr (`habr.md`)
- **Статус:** только для IT-релевантных статей
- **Формат:** Markdown
- **Объём:** полный или близко к полному
- **Примеры тем:** автоматизация процессов, IT-стартапы, продажа технологического бизнеса
- **Пока не актуален** — в контент-плане нет явно IT-статей

## Воркфрейм дистрибуции

### Когда делать
После публикации каждых 5 статей на сайте (или по готовности).

### Шаги

1. **Создать папку** `_distributions/SLUG/`
2. **Написать vc.html** — прочитать оригинал → выбрать 50–60% ключевых разделов → написать в professional-tone → добавить ссылки в финале
3. **Написать dzen.html** — другой угол + личный голос → 60–70% → ссылки
4. **Написать tenchat.txt** — 400–600 символов, один главный тезис + 3–5 пунктов
5. **Публиковать** по очереди: VC.ru → Дзен → TenChat

### Интервал публикаций
Не публиковать все 3 платформы в один день. Оптимально: VC.ru и Дзен с разницей 2–3 дня, TenChat — отдельно.
