# Таск-менеджер: Сайт a-invest.pro

> **Правила работы с таск-менеджером**
>
> Любые изменения в этом файле — ТОЛЬКО через подтверждение Саши в чате.
> Никаких самостоятельных правок — ни добавлений, ни удалений, ни переформулировок.
>
> Порядок работы:
> 1. Показать правки в чате (что добавить / изменить / удалить)
> 2. Получить подтверждение Саши
> 3. Внести изменения в файл
>
> - **Добавить:** формулируем вместе → Саша подтверждает → вносим
> - **Изменить:** показываем изменение в чате → подтверждение → вносим
> - **Удалить:** Саша говорит что выполнено → подтверждаю что удалить → удаляю
> - Статьи пишем парами; следующая пара появляется только после завершения текущей
> - Выполненные задачи удаляются, не отмечаются
> - Даты всегда абсолютные (не «через неделю», а «2026-03-19»)

## Регулярные
> Повторяющиеся задачи с описанием и частотой

**Производство статей — 10-12 в неделю**
Следуем content-guide.md. Claude Code пишет, Саша ревьюит. Каждая статья = правильные внутренние ссылки, обновление sitemap, blog.html, related.js, llms.txt. Цель: все 100 статей до конца весны 2026.
*Частота: ежедневно, ~2 статьи в день. Всегда показываем в задачах актуальную пару, но не поднимаем задачу по след. статьям, пока по текущим не отмечена выполненной.*

**Дистрибуция на платных площадках — 8-10к/мес**
VC.ru, Habr, Pikabu. Купить подписку → выложить 4-6 статей → ссылочная масса. Саша управляет аккаунтами, Claude Code форматирует.
*Триггер: начать при 30+ статей. Частота: 1 раз в месяц*

**Search Console — ежемесячный разбор**
4 корзины: низкий CTR, позиции 8-20, неправильный интент, смежные запросы. На основе данных: переписывать title/description, добавлять/расширять статьи. Саша смотрит данные, Claude Code вносит правки.
*Триггер: с мая 2026 (нужно 2-3 мес для накопления данных). Частота: ежемесячно*

**Крупные SEO-задачи — 1 в неделю**
Когда статейная машина выйдет на 30-40 статей, параллельно со статьями делать по одной крупной задаче в неделю из отложенных/бэклога.
*Триггер: при 30-40 статей (~конец марта). Частота: еженедельно*

**Трекинг органического трафика — ежемесячно**
Сравнить фактические визиты из Яндекс.Метрики с baseline-прогнозом (_materials/seo-forecast-2026.html). Фиксировать отклонение. При отставании >40% — анализ причин и корректировка стратегии.
*Контрольные точки (baseline): июнь — 300, сентябрь — 1,500, декабрь — 3,500*
*Красная зона: июнь < 200, сентябрь < 700, декабрь < 500*
*Частота: 1 раз в месяц, начало — май 2026*
