## Воркфрейм: написание новой статьи

### Шаг 0 — прочитать бриф
Открыть `_materials/brief.md`. Убедиться в понимании текущего этапа и задач.

### Шаг 1 — выбрать статью
Открыть `_materials/content-plan.md`, выбрать ⬜ статью с 🔴 приоритетом.

### Шаг 2 — подготовить материал
- Посмотреть TG-посты и VC по теме
- Выписать цифры, кейсы, формулировки Александра

### Шаг 3 — написать по шаблону
Скопировать `_template.html` → `blog/SLUG.html`
Следовать инструкции внутри шаблона (ШАГ 1–5).

### Шаг 4 — чеклист публикации
- [ ] Карточка добавлена в `blog.html`
- [ ] URL добавлен в `sitemap.xml`
- [ ] Перелинковка: ссылка поставлена в 1–2 существующих статьях
- [ ] Список статей обновлён в `_template.html`
- [ ] Статус в `_materials/content-plan.md` → ✅

### Шаг 5 — деплой
```bash
cd "/Users/sashafyc/Desktop/A-INVEST/Сайт"
git add blog/SLUG.html blog.html sitemap.xml
git commit -m "Добавлена статья: ЗАГОЛОВОК"
git push
```
